JS正则表达式复习笔记

发表于 2017-11-05 | 分类于 Learning | | 阅读次数:

前言

温故而知新，可以为师矣。对于正则表达式的学习最好的方法还是多练多回顾，复习，走起！

正则表达式（英文：Regular Expression，在代码中常简写为regex、regexp、RE）使用单个字符串来描述、匹配一系列符合某个语法规则的字符串搜索模式。它可用于所有文本搜索和文本替换的操作。简而言之，它就是通过计算机认识的语言制定的规则，能让计算机帮我们找到对应的东西。

正文

定义正则

  1. var re = new RegExp("a"); //RegExp 对象，参数就是我们想要制定的规则
  2. var re = /a/; //简写方法（推荐使用，性能更好，不能为空）

正则的常用方法

test():

在字符串中查找符合正则的内容，若查找到返回true，反之则返回false。

用法：正则.test(字符串)

例子：判断是否是数字 

var str = "4396";  
var re = /\D/;  //  \D\代表非数字
if (re.test(str)){
    alert("不全是数字");
} else{
    alert("全是数字");
}

正则表达式中有很多符号，代表着不同的意思。

  • \s : 空格
  • \S : 非空格
  • \d : 数字
  • \D : 非数字
  • \w : 字符（字母、数字或者下划线）
  • \W : 非字符
    （不罗列了，下面会根据例子依次讲一些常用的字符）

search():

在字符串搜索符合正则的内容，搜索到就返回出现的位置（从0开始，如果匹配的不只是一个字母，那就只会返回第一个字母的位置），如果搜索失败就返回-1。

用法：字符串.search(正则)

NOTE: 正则中默认区分大小写，如果想不区分大小写，就在正则的最后加修饰符i（ignore的缩写）。

例子：在字符串中查找字母b，且不区分大小写

var str = "abcdf";
var re = /B/i; //也可以写成 var re = new RegExp('B','i');

alert(str.search(re));  //1

match():

在字符串中搜索符合规则的内容，搜索成功就返回内容，格式为数组，失败就返回null。

用法：字符串.match(正则)

量词：+代表至少匹配一次

全局匹配：g（global的缩写，正则中默认，只要搜索到符合规则的内容就会结束搜索）

例子：找出指定格式的所有数字 

var str = "rng1skt2rng3skt45";
var re = /\d/g;  //每次匹配至少一个数字且全局匹配，如果不是全局匹配，当找到数字1，它就会停止。
加上全局匹配就会从开始到结束一直去搜索符合规则的。如果没有加号，匹配的结果就是[1,2,3,4,5] ，这
并不是我们想要的。
alert(str.match(re));  //[1,2,3,45]

replace():

查找符合正则的字符串，就替换成对应的字符串。返回替换后的内容。

用法：字符串.replace(正则，新的字符串/回调函数)（在回调函数中，第一个参数指的是每次匹配成功的字符）

例子：敏感词过滤。 

var str = "我爱北京天安门，天安门上太阳升";
var re = /北京|天安门/g;
var str2 = str.replace(re,'*');
alert(str2)  //我爱**，*上太阳升
//这种只是把找到的变成了一个 * ，并不能几个字就对应几个 * 。

要想实现几个字对应几个 * ，我们可以用回调函数实现。 

var str = "我爱北京天安门，天安门上太阳升"
var re = /北京|天安门/g;
var str2 = str.replace(re,function(str){
alert(str); //用来测试：函数的第一个参数代表每次搜索到的符合正则的字符，所以第一次str指的是北京，
第二次str是天安门，第三次str是天安门
var result = '';
for(var i=0;i<str.length;i++){
    result += '*';
}
    return result;
});
alert(str2)  //我爱*****，***上太阳升

正则中的字符

(): 小括号，叫做分组符。相当于数学里面的括号。

var str = '2013-6-7';
var re1 = /\d-+/g;  //全局匹一个数字，横杠，横杠数量至少为1，匹配结果为：['3-','6-']
var re2 = /(\d-)+/g;  //全局匹配数字，横杠，数字和横杠整体数量至少为1，匹配结果为：['3-6-']
var re3 = /(\d+)(-)/g;  //全局匹配至少一个数字，匹配一个横杠，匹配结果为：['2013-','6-']

正则中的每一个带小括号的项，都叫做这个正则的子项。

例子：让2013-6-7变成2013.6.7 

var str = '2013-6-7';
var re = /(\d+)(-)/g;
str = str.replace(re,function($0,$1,$2) {
//replace()中如果有子项，
//第一个参数：$0 (匹配成功后的整体结果 2013-, 6-)，
//第二个参数：$1 (匹配成功后的第一个分组，这里指的是 \d ，2013, 6 )，
//第三个参数：$2 (匹配成功后的第二个分组，这里指的是 - ，-, -)。
    return $1 + '.';
});
alert(str);

match 方法也会返回自己的子项，如下：

var str = 'abc';
var re = /(a)(b)(c)/;
alert(str.match(re)); //[abc,a,b,c]
//返回的是匹配结果以及每一个子项，当 match 不加 g 的时候才可以取到子项的集合

补充:exec() 方法和 match() 方法类似，搜索符合规则的内容，并返回内容，格式为数组。

用法：正则.exec(字符串)

属性：input //代表要匹配的字符串；index //每次匹配成功的字符串中第一个字符的位置

例子一：

var testStr = "now test001 test002";
var re = /test(\d+)/; //不是全局匹配，只匹配一次
var r = re.exec(testStr);
console.log(r);
//['test001','001',index:4,input:'now test001 test002']
//返回了包含匹配结果、子项、index属性、input属性的数组
//另外 r.length = 2

例子二：

var testStr = "now test001 test002";
var re = /test(/d+)/g; //全局匹配
var r = re.exec(testStr);
console.log(r);
//['test001','001',index:4,input:'now test001 test002']
//结果为啥一样，我用 while 循环，可一直匹配出这个结果，并没有接着上次匹配的位置开//始匹配。

[]:表示某个集合中的任意一个，比如[abc]整体代表一个字符，匹配 a b c 中的任意一个，也可以是范围，范围必须从小到大[0-9]。

[^a]:整体代表一个字符，^写在[]里面的话，就代表排除的意思。

例子：匹配 HTML 标签，比如<div class="b">hahaha</div>找出标签<div class="b"></div>

var re = /<[^>]+>/g; 
//匹配左括号，中间至少一个非右括号的内容（标签里面的一些东西），然后匹配右括号
var re = /<[\w\W]+>/g;
//匹配左括号，中间至少一个字符或者非字符的内容，然后匹配右括号

转义字符

除了前面提到的六个还包括：

. ：任意字符

\.: 真正的点

\b ：独立的部分（起始，结束，空格）

\B : 非独立的部分

关于最后两个我们来看个例子：

var str1 = "onetwo";
var str2 = "one two";
var re = /one\b/; //e 后面必须是独立的
console.log(re.test(str1)); //false
console.log(re.test(str2)); //true

量词:代表出现的次数

{n,m}: 至少出现 n 次，最多 m 次

{n,}: 至少 n 次

* ：任意次，相当于{0,}

? : 零次或者一次，相当于{0，1}

+ ：至少一次，相当于{1,}

{n}；正好 n 次

$放在正则的最后位置，就代表结束的意思

常用的一些表单校验

匹配中文：[\u4e00-\u9fa5] //中文 ACALL 码的范围

行首行尾空格：^\s*|\s*$ //首行出现任意个空格或者尾行出现任意个空格

Email：^\w+@[a-z0-9]+(\.[a-z]+){1,3}$

//起始至少为一个字符，然后匹配@，接着为任意小写字母或者数字，
\.代表真正的点，.后面为至少一个字符（a-z），
同时这个（比如 .com ）整体为一个子项作为结束，可以出现1-3次。

网址：[a-zA-Z]+://[^\s]* //匹配不分大小写的任意字母，接着是//，后面是非空格的任意字符

邮政编码：[1-9]\d{5} //起始数字不能为0，然后是5个数字

身份证：[1-9]\d{14}|[1-9]\d{17}|[1-9]\d{16}x
