2018年3月21日面试总结

发表于 2018-03-21 | 分类于 Interview | | 阅读次数:   |   字数统计: 940(字)   |   阅读时长: 3(分)

你可以走的慢一些，但绝不能停下脚步

前言

做下自我介绍吧！

你为什么要选择前端？

你学习途径是什么？你都看过哪些书？

你觉得你学的最好的是哪一方面？

你大学没有学过数据库吗？

jQuery 当然要学！

作品什么的不重要，我又不能确认是不是你自己写的。

来，写段代码吧！

你进入社会招聘了，可不是像毕设那样随随便便就糊弄过去了。

你还有什么想问我的吗？

你要是真的喜欢，估计再学个两三个月可以找到工作。

你可以走了，有什么问题问 HR 吧，等她的通知。

正文

这次面试和上次不一样的地方是需要手写代码，对于我来说还是生疏了，写出来的代码不规范。对于函数的调用，以及一些方法的准确性，我不能保证写出来的代码一定能够跑起来，他让我写的那段代码现在想起来真的很简单，我面试回来后在电脑上就实现了，可是当场写为什么就写不好呢？这个真的需要反思。

我为什么选择前端？可能面试官知道你是个转行的人之后都会问这个问题。考研失败后我就着手转行了，有些人会冷嘲热讽道：“怎么本专业都学不好的人，会以为自己能在互联网行业赚到高薪”。首先，我得承认令我转行的因素里互联网行业的高薪占不小的一部分。其次，我本身真心不喜欢化学，我不想解释原因。既然不喜欢，我就没必要还待在这个坑里。于是我就开始在网上寻找各种转行的资料，一开始学了几天 python ，因为乎友们都说这是最好的入门语言。但是我一搜网上的 python 工作岗位不是很多，与此同时我发现前端很火，并且看见很多人说前端好入门，我就一脚踏入前端，之后就再也没有没有改变过。最后我想说性格使然，我本身比较内向，不爱和陌生的人说话，我觉得倒饬倒饬代码，学门安生立命的技术更有成就感和安全感。而且学好了至少在薪水确实能给你相应的回报。

面试官不会想听这么多，那我就总结成几点。

  1. 本身就是不喜欢化学。
  2. 我的性格让我选择做一名程序员。
  3. 前端刚开始对新手比较友好，一边写一边看页面很有成就感，有学下去的动力。
  4. 前端的知识所有大学都不教，大家都是自学，只是时间问题，时间长了，别人会的你也能学会。

关于我看了哪些书和教程，例举一下：

HTML: w3school
CSS: w3scool、《CSS揭秘》、《HTML5和CSS3权威指南》下册（这本书和大漠的《图解CSS3》封面很像，我买错了，这书不是很好）
JavaScript: 《JavaScript和DOM编程艺术》、《JavaScript高级程序设计》、《JavaScript语言精粹》
Vue: Vue 官方文档
HTTP: 《图解HTTP》

后记

我投了这么多次简历发现 jQuery 还是很多公司要求熟悉的库，我以前看知乎说这玩意已经被淘汰了不用去看了，于是我看了《锋利的jQuery》前几章就扔掉去看 Vue 官方文档了。现在发现依然很坚挺嘛，今晚我就去看。

革命尚未成功，同志仍需努力。

补一个问题：“innerHTML 和 innerText 有什么区别？”
